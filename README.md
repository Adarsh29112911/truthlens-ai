 📰 TruthLens AI
AI-Powered Fake News Detection System

TruthLens AI is a Machine Learning-based web application that detects whether a news article is **Real or Fake** using Natural Language Processing (NLP).

Developed during **OPS 48 Hackathon** by **Team AAR Intelligence**.

## 🚀 Problem Statement

The rapid spread of misinformation across digital platforms makes it difficult for users to verify the authenticity of news content. Fake news influences public opinion, creates confusion, and reduces trust in information sources.

We aim to build an intelligent system that can automatically analyze and classify news articles in real-time.

💡 Our Solution

TruthLens AI uses NLP techniques and Machine Learning algorithms to:

- Analyze textual patterns in news articles
- Classify content as **Real or Fake**
- Provide a **confidence score**
- Display probability distribution graph

The system is lightweight, fast, and user-friendly.


🧠 Technical Approach

 1️⃣ Dataset

 2️⃣ Text Processing
